Post-development:
Install packaging tools: pip install --upgrade setuptools packaging


Compilation: 
python -m compileall src or "python -m compileall path/to/project/src" from outside folder
Build package by running setup: python setup.py sdist bdist_wheel
Copy the following folders to MSPUserDataTool
   .ipynb_checkpoints
   __pycache__
   certificate
   .env
   userConfig
   userProgram
   utils-1.0.0

All user scripts must import utilities using:
from langodata.utils import read_data, read_profile

Tests:
Test specific util module: eg auth_token.py
Test script: cd MSPUserTests  then python -m unittest test_utils.py

Running in user accessed folder:
Installation: pip install utils-1.0.0-py3-none-any.whl
